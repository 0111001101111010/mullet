## The Mullet Stack

www.mullet.io

[Facebook](http://facebook.github.io/react/) in the front. [Walmart](http://walmartlabs.github.io/hapi/) in the back.

All sitting on top of [Node.js](http://nodejs.org/).

### Get Started

* Install [Node.js](http://nodejs.org/)
* Install npm packages
```
npm install
```
